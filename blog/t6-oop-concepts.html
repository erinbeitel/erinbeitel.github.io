Inheritance vs Composition

Concepts Compared

July 5, 2015

Inheritance indicates the features that one class gets from another. If one class inherits from another then it inherits the accessible methods and variables of that class.

Composition is a more explicit way to obtain the allow access to the same information in a different class as you would if the class was inheriting from another class. With composition, you call the original class instead of implicitly allowing the second class to inherit information.
