Enumerables

Use Cases of the "cycle" Method

June 14, 2015

Enumerables are a collection of classes with methods related to searching, sorting and traversing elements of an array or hash. In each class, successive elements of a collection are made available through the implementation of the each method.

The method cycle - an example of one such method from the enumerable mixin.

cycle calls the block of code given, for each element, a speficied number of times, cyclically. For example, if cycle is called on the range (1...3) and specified to undergo two cycles, for the block of code puts x*x, the method would square each element in the range and then cycle back through and square each element again. The results of this process would be collected into an internal array and printed as 1 4 9 1 4 9 with a return of nil.

(1..3).cycle(2) {|x| puts x*x}

=> 1, 4, 9, 1, 4, 9

=> nil

If a specification is not made for the number of cycles, then the method will continue forever.

(1..3).cycle {|x| puts x*x}

=> 1, 4, 9, 1, 4, 9, 1, 4, 9, 1, 4, 9, 1, 4, 9, 1, 4, 9, 1, 4, 9, 1, 4, 9 forever...

This infinite loop can be broken by tripping a break statement under certian circumstances. For example, a method could cycle through the months of the year and break when the month of May is encountered like so:

months_year.cycle do |mo|

puts mo

break if mo == "May"

end

=> Jan, Feb, Mar, Apr, May

=> nil

More information on methods like cycle can be found in the Ruby doc for the Enumerable mixin.
